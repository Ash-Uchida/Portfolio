Zoom Zone

The Goal: To build an automated scheduling tool that eliminates the manual work of creating meeting links and checking calendar availability.

The Solution: Zoom Zone serves as a centralized hub for bookings. It leverages the Zoom API and Google Calendar API to ensure that every booking is instantly reflected in the user's schedule with a unique meeting URL generated automatically.

Technical Stack: Backend: Vercel (Serverless), Supabase (Database & Auth)

APIs: Google Calendar (OAuth 2.0), Zoom API

Frontend: React/Next.js
